دليل إتقان نظام ECC

الفهرس الشامل (Table of Contents) للدروس العملية لإتقان أدوات الذكاء الاصطناعي المتقدمة.

  • الدرس 01: مقدمة إلى ECC أساسيات
    ما هو النظام؟ وكيف يختلف عن استخدام Claude العادي.
  • الدرس 02: الإعداد العالمي والتبديل بين النماذج
    كيفية التثبيت، إعدادات خوادم MCP، والتبديل الاستراتيجي بين (Opus, Sonnet, Haiku).
  • الدرس 03: المشاريع الجديدة وهيكلة المجلدات
    استخدام الوكلاء لبناء مشاريع جديدة مثل SaaS بالـ Laravel و Next.js، وفهم استراتيجيات مجلد .claude/.
  • الدرس 04: دمج النظام بالمشاريع الحالية
    كيف تنظف وتطور مشروعاً قديماً (Legacy Code) دون تدميره.
  • الدرس 05: إضافة الميزات الجديدة و TDD تطبيق عملي
    كتابة الاختبارات أولاً ثم التطوير، مع مثال إضافة نظام تسجيل الدخول عبر GitHub.
  • الدرس 06: احتراف الوكلاء (Agents)
    التعرف على الوكلاء المعماريين، محللي الأخطاء (Build Resolvers)، والمراجعين.
  • الدرس 07: إتقان المهارات (Skills)
    تنفيذ مهام البحث العميق والتقيد بالمعايير الفنية الجاهزة.
  • الدرس 08: فهم الخطافات (Hooks)
    أتمتة الأمان وتنسيق الكود، ومنع تسريب البيانات السرية قبل حفظها.
  • الدرس 09: الإضافات المخصصة متقدم
    كيف تنشئ وكلاء ومهارات وخطافات تناسب فريق عملك المحلي حصراً.
  • الدرس 10: بناء أنظمة التصميم (Design Systems)
    تأسيس هوية بصرية صارمة للتطبيق باستخدام وكيل التخطيط ومهارات الـ Frontend.
  • الدرس 11: أفضل الممارسات اللغوية
    إجبار النظام على اتباع معايير محددة لأي لغة مثل Python (Django) أو Go أو Rust.
  • الدرس 12: دورة حياة المشاريع المتقدمة
    من التخطيط الاستراتيجي إلى عزل فروع Git وحتى إنشاء ملخص الـ PR النهائي.
  • الدرس 13: تعدد النماذج والوكلاء (Orchestration)
    استخدام Claude و Gemini معاً في مشروع واحد للمراجعة المتبادلة (Peer Review).
  • الدرس 14: الواجهة الرسومية (ECC Dashboard)
    تصفح وإدارة نظام الذكاء الاصطناعي الخاص بك عبر تطبيق واجهة مستخدم رسومية (GUI).
