# الدرس الرابع عشر: استخدام لوحة تحكم ECC (Dashboard)

  الدرس الرابع عشر: استخدام لوحة تحكم ECC (Dashboard)  

## ما هي لوحة تحكم ECC؟

بالرغم من قوة واجهة سطر الأوامر (CLI) الخاصة بـ Claude Code، إلا أن تتبع 64+ وكيل و 260+ مهارة وعشرات الخطافات عبر الطرفية قد يكون مرهقاً. لذلك، يأتي نظام ECC مزوداً بـ **تطبيق سطح مكتب (Desktop GUI)** مبني باستخدام Python/Tkinter يتيح لك استكشاف المكونات وإدارتها بصرياً بكل سهولة.

## كيفية فتح لوحة التحكم (Setup & Open)

تشغيل لوحة التحكم بسيط للغاية. تأكد من أنك متواجد في مسار المشروع الذي يحتوي على ملفات ECC، ثم قم بتشغيل أحد الأمرين التاليين من الطرفية:

### عبر Node.js (المفضل):

```
npm run dashboard
```

### عبر Python مباشرة:

```
python3 ./ecc_dashboard.py
```

ستظهر لك نافذة تطبيق رسومية أنيقة تدعم تبديل السمة (Dark/Light Mode) ותخصيص الخطوط.

## أهم ميزات لوحة التحكم (How to Control)

توفر الواجهة الرسومية تحكماً بصرياً كاملاً في نظامك التشغيلي للذكاء الاصطناعي:

1.  **مستكشف المهارات والوكلاء (Skills & Agents Explorer):** يمكنك استعراض جميع المهارات المتاحة لك، قراءة وصفها (Description)، ومعرفة كيفية استدعائها بنقرة زر دون الحاجة لفتح ملفات `.md` يدوياً.
2.  **إدارة خوادم MCP:** يمكنك من خلال لوحة التحكم رؤية الخوادم المفعلة حالياً ومراقبة حالتها (Status).
3.  **مراقبة الخطافات (Hooks):** يمكنك استعراض الخطافات المفعلة (PreToolUse و PostToolUse) والتأكد من أنها لم تتكرر بالخطأ بسبب عمليات التثبيت المتعددة.

## الواجهة القادمة (ECC 2.0 Rust Dashboard)

إذا كنت تعمل على النسخة المستقبلية التجريبية (Alpha) من ECC، ستلاحظ وجود محرك مبني بلغة **Rust** في مجلد `ecc2/`. هذا المحرك سيقدم أدوات تحكم إضافية فائقة السرعة للوحة التحكم:

```
# أمثلة لأوامر محرك Rust القادم للوحة التحكم
ecc2 dashboard # فتح اللوحة المتطورة
ecc2 sessions  # إدارة الجلسات المفتوحة
ecc2 daemon    # تشغيل النظام في الخلفية
```

## الخلاصة

استخدم **الطرفية (CLI)** عندما تكتب الكود الفعلي (Coding Phase). واستخدم **لوحة التحكم (Dashboard)** عندما تريد الاستكشاف (Exploration)، مراجعة المهارات المتاحة، أو مراقبة حالة نظامك (System Monitoring).
